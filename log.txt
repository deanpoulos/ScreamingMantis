Learning:
    - `pip` is a package manager for python related plugins/wrappers
    - a function `wrapper` is code to make complex functions more accessible
    - to interact with HTML, we need to import the `requests` library
        > r = requests.get('https://api.github.com/user', auth=('user', 'pass'))
        > r.status_code
          =200
        > r.headers['content-type']
          ='application/json; charset=utf8'
        > r.encoding
          ='utf-8'
        > r.text
          =u'{"type":"User"...' 
        > r.json() 
          ={u'private_gists': 419, u'total_private_repos': 77, ...}
        # this is what an interpreter would return
